PDF 뷰어

HTML5 표준 태그 <iframe>를 활용한 PDF 렌더링 입니다.

💡
준비하신 PDF 파일은 src/main/resources/static/pdf/sample.pdf 경로에 업로드하시면 아래 뷰어 화면에 자동으로 렌더링됩니다.

🌐 <iframe> 태그 분석

웹 문서 내에 또 다른 HTML 문서를 내장할 때 사용하는 표준 태그입니다. PDF 출력 시 가장 널리 사용됩니다.

  • 장점: 사용이 단순하며 모던 브라우저 대부분에서 내장 리더 툴바와 함께 정상 작동합니다.
  • 단점: 모바일 환경이나 구형 브라우저에서 인라인 표시 대신 다운로드 창이 뜨거나 빈 화면이 나타날 수 있습니다.
